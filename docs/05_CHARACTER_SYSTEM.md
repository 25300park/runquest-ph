# Character System

## Project Name

**RunQuest PH**

## Purpose

This document defines the character system for RunQuest PH.

The character system is inspired by RPG-style character selection, but it is adapted for walking, jogging, running, and local exploration.

The goal is not to copy a combat game structure. The goal is to transform the idea of character classes, growth paths, equipment, and skills into a fitness and exploration experience.

---

# 1. Character System Concept

In RunQuest PH, a character is not only a visual avatar.

A character represents the user’s exercise style, motivation, and journey direction.

The user does not simply choose how they look.
The user chooses how they want to grow.

Core concept:

> Choose your running companion. Choose your journey style.

---

# 2. MVP Character Count

The MVP will start with **3 characters**.

The first 3 characters should be simple, memorable, and easy to understand.

Recommended MVP character types:

1. The Explorer
2. The Challenger
3. The Guardian

Each character should have:

* Name
* Visual style
* Short story
* Journey type
* Starting title
* Small bonus
* Future skill tree direction

---

# 3. Character Design Principle

Characters should be:

* Human-style
* Cute
* Friendly
* Motivating
* Beginner-friendly
* Suitable for Philippine local travel
* Expandable into fantasy RPG style later

Characters should not directly label users by body type in a negative way.

The app should avoid language that makes users feel judged by weight, fitness level, body shape, or appearance.

Instead of body type, characters should represent:

* Exercise style
* Motivation style
* Journey type
* Habit pattern
* Exploration preference

---

# 4. Character 1: The Explorer

## Concept

The Explorer loves discovering new places, routes, streets, parks, and hidden corners of the city.

This character is suitable for users who want to make exercise feel like local travel.

## User Type

The Explorer is good for users who think:

* I want to explore new places.
* I like walking and jogging around different areas.
* I enjoy discovering new routes more than chasing records.
* I want fitness to feel like a journey.

## Starting Story

The Explorer begins the journey with a small map, light shoes, and curiosity.

They believe that every street has a story and every route can become an adventure.

## Starting Bonus

Suggested MVP bonus:

* First-time course completion XP +3%
* Area exploration progress +3%

## Future Skill Tree

Possible skill tree direction:

1. Route Discovery
2. Area Mastery
3. Travel Memory

## Future Equipment

Possible equipment:

* Map
* Backpack
* Travel shoes
* Explorer hat
* Route journal

---

# 5. Character 2: The Challenger

## Concept

The Challenger enjoys goals, pace, distance, and personal improvement.

This character is suitable for users who want to improve their running performance.

## User Type

The Challenger is good for users who think:

* I want to improve my pace.
* I want to complete harder routes.
* I enjoy records, badges, and challenges.
* I want to prepare for fun runs or marathons.

## Starting Story

The Challenger begins the journey with running shoes, a training watch, and a clear goal.

They believe every route is a test and every finish line is a step toward becoming stronger.

## Starting Bonus

Suggested MVP bonus:

* Good pace XP +3%
* Hard course XP +3%

## Future Skill Tree

Possible skill tree direction:

1. Pace Training
2. Difficulty Mastery
3. Streak Power

## Future Equipment

Possible equipment:

* Running watch
* Speed shoes
* Medal
* Training band
* Race bib

---

# 6. Character 3: The Guardian

## Concept

The Guardian focuses on steady habits, health, safety, and long-term consistency.

This character is suitable for beginners, comeback users, and users who want a gentle fitness journey.

## User Type

The Guardian is good for users who think:

* I want to start slowly.
* I want to build a healthy habit.
* I do not want too much pressure.
* I want to walk, jog, and improve step by step.

## Starting Story

The Guardian begins the journey with comfortable shoes, a water bottle, and patience.

They believe that small steps repeated consistently can change a life.

## Starting Bonus

Suggested MVP bonus:

* Walking course XP +3%
* Scheduled quest miss penalty reduced once per week

## Future Skill Tree

Possible skill tree direction:

1. Steady Habit
2. Recovery Mindset
3. Safe Runner

## Future Equipment

Possible equipment:

* Water bottle
* Comfort shoes
* Health band
* Recovery towel
* Safety light

---

# 7. Character Balance Rule

The MVP should keep character differences small.

Characters should not create a strong advantage or disadvantage.

Recommended MVP rule:

* Character bonuses should be small.
* Users should be able to change characters later.
* Character choice should feel emotional, not stressful.
* No character should be clearly better than the others.

Suggested bonus range:

* 3% to 5% maximum for MVP

---

# 8. Character and Skill Tree Expansion

The MVP does not need a full skill tree.

However, the database and system should allow future expansion.

Future character features may include:

* Skill points
* Passive skills
* Character equipment
* Premium skins
* Area-based costumes
* Event costumes
* Fantasy class upgrades
* Watch-verified skills

Example future upgrade:

The Explorer → Island Pathfinder
The Challenger → Marathon Knight
The Guardian → Wellness Sage

---

# 9. Character Selection Flow

After registration, the user selects one character.

Suggested screen message:

> Choose your running companion.
> Every journey starts differently.
> Pick the character that feels like you today.

The character selection screen should show:

* Character image
* Character name
* Short story
* Journey type
* Small starting bonus
* Recommended for text

Example:

Recommended for:

* Beginners
* Explorers
* Challenge seekers
* Comeback runners

---

# 10. Long-Term Vision

The character system can become one of the main reasons users stay with the app.

Long-term character value:

* Emotional attachment
* Game progression
* Premium skins
* Partner-branded items
* Event rewards
* Area completion costumes
* Fantasy RPG expansion

The character should become the visual expression of the user’s fitness journey.
